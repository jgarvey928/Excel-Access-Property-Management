## 📖 About the Project

Developed by John Garvey, this project is a comprehensive property management tool built entirely using Microsoft Office Visual Basic for Applications (VBA). The application is designed to streamline property management tasks through an intuitive graphical user interface (GUI).

**Key Storage Features:**
* **Database Mode:** Stores property data efficiently in an MS Access database (`.accdb`).
* **Legacy Mode:** Capable of storing data directly within Excel spreadsheets.

---

## ⚙️ System Requirements

To ensure the application runs smoothly, your system should meet the following specifications:
* **Operating System:** Windows 11 x64 (Tested on build 10.0.22631)
* **Software:** Microsoft Excel and Microsoft Access (Desktop versions supporting VBA)

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

### 1. Installation
To run this application, you must download both the Excel macro-enabled workbook and the Access database file:
1. Download `EAPM.xlsm`
2. Download `EAPM.accdb`
3. ⚠️ **Important:** Place *both* files in the exact same local folder on your PC.

### 2. Execution & Credentials
1. Open the `EAPM.xlsm` file.
